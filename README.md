- **Clinical Decision Support** вЂ” Aggregated insights and clinical suggestions for Admins/Providers, with an explicit disclaimer that it assists вЂ” not replaces вЂ” professional judgment.
- **Feedback System** вЂ” User, Doctor, and Suggestion feedback stored in the database.
- **Admin Panel** вЂ” Dataset upload, user management, statistics, model training/monitoring, report management, and feedback review.

## Setup & Running

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt --break-system-packages
   ```

2. **(If the dataset isn't present) generate the synthetic dataset:**
   ```bash
   python data/generate_dataset.py
   ```

3. **Run the app:**
   ```bash
   streamlit run app.py
   ```

4. **Register an account** (choose Patient, Healthcare Provider, or Admin) and log in.

5. **Train the models** from the Admin/Provider "Model Training" page before
   using "Diabetes Prediction" for the first time.

## Dataset

This project targets the schema of the Kaggle
[Diabetic Prediction dataset](https://www.kaggle.com/datasets/akhilalexander/diabeticprediction)
(`age`, `gender`, `bmi`, `hypertension`, `heart_disease`, `smoking_history`,
`blood_glucose_level`, `HbA1c_level`, `diabetes`). Since direct dataset
download wasn't available in the build environment, a clinically plausible
**synthetic dataset of 8,000 records** matching this exact schema is
generated by `data/generate_dataset.py` вЂ” simply replace `data/diabetes.csv`
with the real Kaggle file (same column names) to use real data instead.

## Security Notes

- Passwords are hashed with bcrypt (never stored in plain text).
- All database queries are parameterized (SQL-injection safe).
- Role-Based Access Control restricts Model Training, Clinical Decision
  Support, and the Admin Panel to appropriate roles.

## Medical Disclaimer

This system is a screening and educational decision-support tool. It does
**not** provide medical diagnoses and does not replace professional medical
judgment. Always consult a qualified healthcare provider for clinical
decisions.
